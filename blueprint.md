# Project Blueprint

## Overview

This project is a simple, framework-less web application that serves as a lottery number generator. It is built using modern HTML, CSS, and JavaScript, leveraging Web Components for modularity.

## Current State

### Features

*   **Lottery Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Web Component:** Uses a custom element `<lotto-ball>` to display each number.
*   **Modern UI:** A clean and simple interface with a button to generate numbers and a display area for the results.

### Design

*   **Layout:** Centered layout with a clear call to action.
*   **Typography:** Simple and readable font.
*   **Color Scheme:** A playful color scheme with colorful lottery balls.
*   **Interactivity:** A button to trigger the number generation.
